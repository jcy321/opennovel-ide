# OpenNovel IDE

> **OpenNovel 桌面客户端** - 基于 VS Code 二次开发的小说创作 IDE

OpenNovel IDE 是 OpenNovel 项目的桌面客户端，提供专业的小说创作环境，与后端服务 (opennovel-core) 配合使用。

## ✨ 特性

- **群聊协作界面**: 与 8 个 AI Agent 通过群聊方式协作创作
- **资源管理器**: 混合显示书籍、人物、世界观、剧情、伏笔等
- **Agent 状态面板**: 实时查看每个 Agent 的状态和任务
- **知识库浏览**: 可视化浏览人物数据库、世界观数据库等
- **批注系统**: Agent 在章节文件中添加批注，用户可查看和回应
- **流式输出**: 实时显示 Agent 的思考和回复过程

## 🖥️ 系统要求

- Windows 10+ / macOS 10.15+ / Linux (x64)
- 后端服务运行中 (opennovel-core)

## 🚀 快速开始

### 连接后端服务

1. 启动后端服务 (IDE 模式，端口 6688)：

```bash
opennovel
```

2. 启动 OpenNovel IDE，首次使用会显示欢迎页
3. 连接成功后进入创作仪表盘

### 创建新书

1. 点击资源管理器中的 "➕ 创建新书"
2. 输入书名
3. 与规划者 Agent 讨论书籍设定
4. 点击 "构思完成" 进入撰写阶段

| **规划者** | 新书规划、头脑风暴 | 构思中 |
| **天道** | 剧情推演、伏笔管理 | 撰写中 |
| **刘和平** | 人物塑造、一致性维护 | 撰写中 |
| **世界观守护者** | 规则检查、一致性验证 | 撰写中 |
| **执笔** | 内容撰写、批注执行 | 撰写中 |
| **审阅** | 质量评估、阅读体验 | 撰写中 |
| **观察者** | 知识库管理、Agent协调 | 全阶段 |
| **调研者** | 市场研究、爆点分析 | 知识库建立 |
